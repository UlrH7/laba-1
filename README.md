Это программа для заметок на Python с графическим интерфейсом на Tkinter.

Программа делает простые вещи: создаёт заметки, позволяет их редактировать, сохраняет изменения и удаляет ненужные. Все заметки хранятся в файле notes.json, так что после закрытия программы они никуда не пропадают. Ещё есть поиск, который сразу показывает только те заметки, где встречается введённое слово.

Главная изюминка программы — это цвета. Во-первых, можно менять цветовую схему всего интерфейса, нажав на кнопку "Сменить тему". Программа сама случайным образом выбирает одну из пяти тем и перекрашивает панели и заголовки. Во-вторых, у каждой заметки может быть свой цвет фона. Нажал на кнопку "Цвет заметки" или Ctrl+C — и фон заметки меняется на случайный из десяти пастельных оттенков.

Ещё есть возможность экспортировать заметку в файл формата Markdown и импортировать заметку из такого файла. Это удобно, если нужно поделиться заметкой или перенести её в другую программу.

Управление сделано двумя способами: можно тыкать кнопки мышкой, а можно использовать горячие клавиши. Я добавил четыре сочетания: Ctrl+N для новой заметки, Ctrl+S для сохранения, Ctrl+F чтобы перейти в поле поиска, и Ctrl+C чтобы сменить цвет заметки.

Интерфейс устроен так: слева список всех заметок в виде цветных карточек, справа поля для заголовка и содержимого выбранной заметки. Сверху панель с кнопками и поиском. Всё просто и наглядно.
